# 🔎 GitHub User Finder Pro

Aplicação web que permite buscar perfis do GitHub e visualizar informações importantes como seguidores, repositórios e dados do usuário.

## 🚀 Demonstração

👉 https://juslli.github.io/GitHub-User-Finder-pro/

## 📸 Preview

![preview](preview.png)

## 🛠 Tecnologias usadas

* HTML
* CSS
* JavaScript
* GitHub API

## 📌 Funcionalidades

✔ Buscar usuários do GitHub
✔ Mostrar avatar e bio
✔ Mostrar seguidores e repositórios
✔ Listar últimos repositórios
✔ Link direto para o perfil

## 🎨 Interface

Tema **Dark com vermelho rubi**, focado em design moderno.

## 📚 O que aprendi

* Consumo de API com `fetch`
* Manipulação do DOM
* Async / Await
* Estruturação de projetos
